My Projects

Throughout my BBIT degree, I have started multiple entrepreneurial ventures and executed several projects across both business and IT domains. Below are details on three of my key projects:

Project 1: BizConnect

Focus: Business & IT Integration

Tech Stack: Python, Django, SQL

BizConnect is a web portal designed to assist local businesses in transitioning online. It allows business owners to manage inventories, generate electronic receipts, and post technical gigs for local students.

Project 2: SwiftSettle

Focus: Mobile Utility App (Successful Launch)

Tech Stack: Kotlin, SQLite

SwiftSettle is a mobile application developed to help university groups and micro-merchants split shared costs and manage micro-debts. It has been successfully launched on app stores, gaining active users and helping me practice mobile development and release management.

Project 3: SecureAudit

Focus: Cybersecurity & QA Tool

Tech Stack: C++, C

SecureAudit is a lightweight static code analyzer built to parse source code files. It scans for common security vulnerabilities (like buffer overflows) and quality assurance bugs, reflecting my deep interest in cybersecurity and quality control.

Skills Summary

I have built various websites, developed multiple mobile applications, and started several businesses. My primary focus is to continuous learning, aiming to master the technologies I work with to build robust, secure systems.
